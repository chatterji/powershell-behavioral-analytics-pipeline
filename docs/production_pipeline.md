# Production Pipeline

1.  Collect previous day's PowerShell logs.
2.  Load logs into HDFS.
3.  Build analytical DataFrame.
4.  Engineer behavioral features.
5.  Fit regression model.
6.  Rank residuals.
7.  Generate visualization.
8.  Produce daily outlier report for threat hunting.

The pipeline was repeatable, explainable, and suitable for scheduled
execution.
