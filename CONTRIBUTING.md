# Contributing

Thank you for your interest in this project.

Although this repository is maintained as a professional portfolio
project, suggestions and improvements are welcome.

## Ways to Contribute

-   Report documentation issues
-   Suggest enhancements
-   Improve sample code
-   Recommend additional cybersecurity analytics techniques

## Coding Standards

-   Follow PEP 8 for Python
-   Write modular, reusable functions
-   Include clear comments explaining business purpose
-   Prefer explainable analytics over unnecessary complexity

## Pull Requests

Please ensure:

1.  Documentation is updated.
2.  Code is formatted consistently.
3.  Sample data remains synthetic.
4.  No confidential information is introduced.

## Branch Naming

-   feature/`<name>`{=html}
-   bugfix/`<name>`{=html}
-   docs/`<name>`{=html}

Thank you for helping improve this repository.
